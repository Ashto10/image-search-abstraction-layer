Image Search Abstraction Layer

A simple API that takes a search query, and returns a list of image urls, alt texts, and original urls from Imgur. An offset parameter can optionally be passed to paginate through the results.

Alternatively, you can also view previous search queries and when they were made.

joyous-spandex.glitch.me/api/imageSearch?q=<QUERY> joyous-spandex.glitch.me/api/imageSearch?q=<QUERY>&offset=<OFFSET> joyous-spandex.glitch.me/api/latest
